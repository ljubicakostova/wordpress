Simple Body Mass Index Calculator as a Widget on your website. You can customize it as you want to match Your Theme. 

== Description ==

It lets you show a Simple BMI Calculator form as a widget. The result is generated dynamically by JavaScript. 
You can change how it looks like by editing the main.css file in 'plugins/simple-body-mass-index-calculator/css' or using the bmi_calculator class.

#### Widget
* Just insert the widget in your sidebar, type a title, color and change the options you want

#### Try the demo
See the [Simple BMI Calculator demo](http://bmi.znajomek.com "Simple BMI Calculator") in the sidebar at the right

== Installation ==

1. Upload  "simple-body-mass-index-calculator" folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Add the Simple BMI Calculator Widget to your widget enabled space
